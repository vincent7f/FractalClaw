## Overview

FractalClaw is an evolvable agent framework designed based on the Fractal component model, combining core features from OpenClaw.

## Core Features

### 1. Fractal Architecture Features
- **Recursive Component Structure**: Components can be nested to form hierarchical structures
- **Reflection Capability**: Components have introspection and intervention abilities
- **Dynamic Binding**: Flexible binding mechanism between components
- **Execution Model Independence**: Supports multiple execution models

### 2. OpenClaw-style Features
- **Multi-platform Message Routing**: Support for Telegram, Slack, Discord and other platforms
- **Persistent Memory**: Remember user preferences and context
- **Workflow Automation**: Programmable AI workflows
- **Plugin System**: Extensible plugin architecture
- **BYOM (Bring Your Own Model)**: Support for multiple LLM providers

### 3. Self-evolution Capabilities
- **Capability Learning**: Learn new skills from interactions
- **Structure Evolution**: Dynamically adjust component structure
- **Adaptive Optimization**: Auto-optimize based on tasks

## Quick Start

```python
from fractalclaw import FractalAgent, AgentConfig

# Create config
config = AgentConfig(
    name="MyFractalAgent",
    model_provider="ollama",
    model_name="gemma3:12b",
    base_url="http://localhost:11434",
)

# Create agent
agent = FractalAgent(config)

# Initialize and start
agent.initialize()
agent.start()

# Run agent
result = agent.run("Analyze my today's tasks")
print(result)

# Trigger self-evolution
agent.evolve()

# Stop agent
agent.stop()
```
