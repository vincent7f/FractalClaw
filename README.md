## 快速开始 (Quick Start)

```python
from fractalclaw import FractalAgent, AgentConfig

# 创建配置 (Create config)
config = AgentConfig(
    name="MyFractalAgent",
    model_provider="ollama",
    model_name="gemma3:12b",
    base_url="http://localhost:11434",
)

# 创建智能体 (Create agent)
agent = FractalAgent(config)

# 初始化并启动 (Initialize and start)
agent.initialize()
agent.start()

# 运行智能体 (Run agent)
result = agent.run("帮我分析今天的工作任务")
print(result)

# 触发自我进化 (Trigger self-evolution)
agent.evolve()

# 停止智能体 (Stop agent)
agent.stop()
```
